Class Email

Email class.

Note that the view variables ($_viewVars) are set by MeTools\View\EmailView.

Example:

use MeCms\Network\Email\Email;

$email = new Email('default');
$email->to('you@example.com')
    ->subject('About')
    ->send('My message');
